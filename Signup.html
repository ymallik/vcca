Create a CPMS Account

The following information will be used to create an account in the City Payment Management System (CPMS). Only one user can be created per email address. Once an account is created, you will not be able to change the username.

  • Username must be at least six characters long and can have up to five letters and one underscore (_) or period (.) in the middle. Do not use spaces. Examples: John.Doe or John_Doe.
  • Password must be at least eight characters long. It must contain at least one uppercase letter, one lowercase letter, one number and one symbol.
  • Input a phone number using only digits (special characters are not allowed). If you have a US phone number then input your 10-digit phone number, for example: 5690245876. If you have a Non-US phone number then input your country code and phone number, for example: 917648817248.
